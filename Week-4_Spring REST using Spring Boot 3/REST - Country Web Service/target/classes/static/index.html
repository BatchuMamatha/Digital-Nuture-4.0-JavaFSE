Country REST Web Service

Welcome to the Country REST Service demonstration!

Available Endpoints:

Get India Country Details

GET /country

Returns India country details loaded from Spring XML configuration

Sample Request: http://localhost:8083/country

Sample Response: {"code": "IN", "name": "India"}

Get All Countries

GET /countries

Returns all countries loaded from Spring XML configuration

Sample Request: http://localhost:8083/countries

Sample Response: [{"code":"IN","name":"India"},{"code":"US","name":"United States"},{"code":"JP","name":"Japan"},{"code":"DE","name":"Germany"}]

Health Check

GET /country/health

Service health check endpoint

Sample Request: http://localhost:8083/country/health

SME Demonstration Points:

  • Controller Method Explanation: The controller methods load beans from Spring XML configuration and return them as JSON
  • JSON Conversion: Spring Boot automatically converts Java objects to JSON using Jackson ObjectMapper
  • Browser Headers: Open Developer Tools (F12) → Network tab → Make request → View HTTP headers
  • Postman Headers: Make request → Click "Headers" tab → View response headers
