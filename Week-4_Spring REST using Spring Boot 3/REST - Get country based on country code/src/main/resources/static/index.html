Country Code REST Service

Welcome to the Country Code REST Service! This service allows you to get country details by country code with case-insensitive matching.

Main Endpoint:

Get Country by Code (Case Insensitive)

GET /countries/{code}

Description: Returns a specific country based on country code. The country code matching is case insensitive.

Examples:

  • http://localhost:8083/countries/in (lowercase)
  • http://localhost:8083/countries/IN (uppercase)
  • http://localhost:8083/countries/Us (mixed case)
  • http://localhost:8083/countries/jp (Japan)
  • http://localhost:8083/countries/de (Germany)
  • http://localhost:8083/countries/gb (United Kingdom)
  • http://localhost:8083/countries/fr (France)
  • http://localhost:8083/countries/ca (Canada)
  • http://localhost:8083/countries/au (Australia)

Sample Response for India (in/IN/In):

{
  "code": "IN",
  "name": "India"
}

Additional Endpoints:

Get All Countries

GET /countries

Returns all available countries

URL: http://localhost:8083/countries

Health Check

GET /health

Service health check endpoint

URL: http://localhost:8083/health

Implementation Details:

  • Controller: com.cognizant.spring-learn.controller.CountryController
  • Method Annotation: @GetMapping("/countries/{code}")
  • Method Name: getCountry(String code)
  • Service Method: com.cognizant.spring-learn.service.CountryService.getCountry(String code)
  • Path Variable: Country code obtained using @PathVariable
  • Data Source: Countries loaded from country.xml Spring configuration
  • Matching: Case insensitive country code matching using Lambda expressions
  • Response: Returns 200 OK with country details, or 404 Not Found if country doesn't exist

Available Country Codes:

IN - India
US - United States
JP - Japan
DE - Germany
GB - United Kingdom
FR - France
CA - Canada
AU - Australia
